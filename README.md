# AlertaComenzi — Descarcari

AlertaComenzi este o aplicatie desktop usoara pentru monitorizarea comenzilor noi din magazine online.

Aplicatia detecteaza comenzi noi, afiseaza un popup always-on-top, reda un sunet de alerta si trimite notificare in system tray.

Acest repository este utilizat exclusiv pentru distributia aplicatiei (binar Windows). Codul sursa este privat.

## Descarcare

Accesati sectiunea Releases si descarcati ultima versiune disponibila.

## Cerinte sistem

- Windows 10 / 11 (64-bit)

## Utilizare

- Deschideti AlertaComenzi.
- Configurati magazinele si credentialele API din setari.
- Lasati aplicatia pornita pentru monitorizare.
- Cand este detectata o comanda noua, aplicatia afiseaza alerta popup, reda sunetul si trimite notificare in tray.
